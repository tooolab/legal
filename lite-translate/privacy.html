1. Overview

This Privacy Policy describes how Tooolab Inc. ("we," "our," or "us") handles information when you use Lite Translate ("the Extension"), a Chrome browser extension that provides bilingual immersive translation, selection translation, and hover translation.

Lite Translate does not collect, transmit, or store any personal data on external servers. All user data is stored locally in your browser via chrome.storage.local and never leaves your device, except when sending text to translation APIs as described below.

2. Data Stored Locally

All data below is stored exclusively in your browser and is never transmitted to any external server:

DataPurpose
User preferencesLanguage, provider, theme, feature toggles
Glossary termsCustom translation term list
Excluded sitesWebsites where translation is disabled
Translation cacheCached results for performance (auto-expires after 7 days)
UI stateFloating ball position, onboarding flags

3. Translation API Communication

When you initiate a translation, the selected text is sent to the translation API you have chosen. No other data (browsing history, personal information, page URLs, cookies) is sent to any server.

When using third-party APIs (DeepL, OpenAI, Anthropic), their respective privacy policies apply to the text you send.

4. API Key Storage

If you configure a translation provider that requires an API key, it is stored locally in chrome.storage.local. Your API key is:

  • Never transmitted to any server other than the API provider you configured
  • Never shared with us or any third party
  • Deletable at any time by clearing extension data or uninstalling

5. Analytics & Tracking

Lite Translate contains no analytics, telemetry, tracking pixels, or third-party SDKs. We do not track your browsing activity, translation history, or usage patterns.

7. Data Deletion

  • Uninstall the Extension to delete all locally stored data
  • Clear cache from the Extension settings page to remove translation cache
  • Reset defaults from the Extension settings to restore all preferences

8. Children's Privacy

Lite Translate does not knowingly collect any data from children under the age of 13. Since the Extension collects no personal data at all, no special handling is required.

9. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be reflected in the "Effective" date above. Continued use of the Extension after changes constitutes acceptance.
