2. Description of Service

Lite Translate is a free, open-source browser extension that provides bilingual translation for web pages. It offers three translation modes: immersive translation, selection translation, and hover translation.

The Extension supports multiple translation providers including Google Translate, Microsoft Translator, DeepL, OpenAI, Anthropic Claude, Ollama, LM Studio, LibreTranslate, and custom API endpoints.
